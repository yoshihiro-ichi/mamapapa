# README
# 開発言語
- Ruby 2.6.5
- Rails 5.2.5
# 就業Termの技術
- お気に入り機能
- フォロー機能
- devise
- メッセージ機能
# カリキュラム外の技術
- ransack
# 実行手段
```
$ git clone git@github.com:yoshihiro-ichi/mamapapa.gitMamaPapa  
$ cd MamaPapa  
$ bundle install  
$ rails db:create && rails db:migrate  
$ rails s`
```
